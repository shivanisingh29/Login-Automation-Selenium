## Login Automation Selenium
- Automation login flow using Python+Selenium Webdriver

## About The Project 
- This project automates the login process of a web application using Selenium.It includes element detection ,explicit
  & implicit waits,error handling and automatic screenshots.

## Tools & Technologies
- Python
- Selenium
- WebDriver Manager
- ChromeDriver

## Installation
   pip install selenium
   pip install webdriver-manager

## How To Run 
- Pycharm run Process is  shif+F10


## Features
- Automated login using Selenium WebDriver
- Element detection using CSS Selector & XPath
- Implicit & Explicit Waits implemented
- Auto screenshot on success & failure
- Error handling using Try/Except/Finally

## Screenshots
- Screenshot are saved automatically in the project folder
  > login_success.png - Successful login
  > error.png - If any error occurs

## Project Structure
  > Browser open
  > Find element
  > Click actions
  > Waits
  > Screenshots
  > Final login project
  > XPath practice
